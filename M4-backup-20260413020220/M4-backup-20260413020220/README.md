# Project Scaffold

Minimal VS Code workspace scaffold with small Python and Node samples.

Usage

- Run the Python sample:

```bash
python -m src
```

- Or run the Node sample:

```bash
npm install   # optional, no deps by default
npm start
# or: node src/index.js
```

- Run tests (if you add a test runner):

```bash
# Example: pytest
pytest
```

Files created

- README, .gitignore, LICENSE
- `src/` with Python and Node samples
- `tests/` with a placeholder test
- `.vscode/settings.json`
